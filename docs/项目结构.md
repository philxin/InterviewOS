# 📦 InterviewOS 项目结构（最终确认版）

interviewos/  
│  
├── backend/            # Spring Boot (Maven)  
│  
├── frontend/           # Vue 3 + Vite  
│  
├── docs/               # 项目文档  
│  
├── docker/             # Docker 相关配置（可选）  
│  
├── .gitignore  
├── README.md  
└── LICENSE (可选)

* * *

# ✅ 说明

### 1️⃣ backend/

* 独立 Maven 项目

* 单体应用

* 提供 REST API


* * *

### 2️⃣ frontend/

* 独立 Vue + Vite 项目

* 单页应用（SPA）

* 通过 `/api` 调用后端


* * *

### 3️⃣ docs/

用于存放：

* 产品设计文档

* 架构说明

* 数据库设计

* 版本规划


* * *

### 4️⃣ docker/（可后期添加）

未来可能包含：

* PostgreSQL

* Redis

* 应用服务


* * *

# 🧠 当前结构特点

✔ 单仓库（便于个人维护）  
✔ 前后端完全独立  
✔ 可随时拆分为多仓库  
✔ 可扩展 CI/CD
