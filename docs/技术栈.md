# 一、后端技术栈

## ✅ 基础环境

| 技术 | 版本 |
| --- | --- |
| Java | 21 |
| Maven | 3.9+ |
| Spring Boot | 3.3.x |
| Spring AI | 1.0.x（稳定版） |

> 建议使用 Spring Boot 3.3.x，生态成熟稳定。

* * *

## ✅ Spring Boot 核心依赖

| 组件 | 说明 |
| --- | --- |
| spring-boot-starter-web | REST API |
| spring-boot-starter-data-jpa | ORM |
| spring-boot-starter-validation | 参数校验 |
| spring-boot-starter-security | 用户认证（可后期开启） |
| spring-boot-starter-data-redis | 缓存 + Stream |
| spring-boot-starter-test | 单元测试 |

* * *

## ✅ AI 层

通过 **Spring AI** 统一封装 LLM 接入。

支持外部模型接口：

* OpenAI

* Claude

* DeepSeek

* 自定义 HTTP LLM Provider


推荐保留：

| 依赖 | 说明 |
| --- | --- |
| spring-ai-openai-spring-boot-starter | OpenAI 兼容接口 |
| spring-ai-core | Prompt & Chat 抽象 |

这样可以做到：

> 后期自由切换模型，而无需重构业务层。

# 三、InterviewOS 架构定位

这是一个：

* 单体后端（模块化设计）

* 外部 LLM 可插拔

* PostgreSQL + 向量能力

属于：

> AI 驱动的面试认知训练系统（Product-level 架构）
