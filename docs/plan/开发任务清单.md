## 🎯 每日工作建议

### 工作流

1. **开始前**: 拉取最新代码，确保环境正常
2. **开发中**: 遵循架构约束，及时提交代码
3. **结束前**: 自测功能，更新任务清单

### 提交规范

```
feat: 实现知识点创建功能
fix: 修复掌握度计算错误
docs: 更新 API 文档
test: 添加 Service 层单元测试
refactor: 重构 PromptTemplate
```

## 📞 问题排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| LLM 调用超时 | API Key 错误/网络问题 | 检查环境变量和网络 |
| 数据库连接失败 | PostgreSQL 未启动 | 启动数据库服务 |
| 前端跨域错误 | CORS 未配置 | 添加 CORS 配置 |
| 掌握度计算错误 | 公式实现问题 | 检查 updateMastery 方法 |
| JSON 解析失败 | LLM 返回格式异常 | 添加容错处理 |
