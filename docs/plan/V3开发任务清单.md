# InterviewOS V3 开发任务清单

## 📋 任务概览

本清单基于以下文档拆解：

- [V3版本规划.md](./V3版本规划.md)
- [V3数据库设计.md](./V3数据库设计.md)
- [V3接口设计.md](./V3接口设计.md)

任务拆分原则：

1. 先完成 P0 主链路
2. 先做“文件级知识库 + 检索增强训练”，后做“自动知识点识别”
3. 不破坏 V2 已稳定的训练主链路
4. 安全、幂等、可观测性不作为补丁项，而是首批设计内容
