# InterviewOS V4 开发任务清单

## 1. 任务概览

本清单基于以下文档拆解：

- [V4版本规划.md](./V4版本规划.md)
- [V4前端架构设计.md](./V4前端架构设计.md)
- [V4执行看板.md](./V4执行看板.md)

任务拆分原则：

1. 先完成 `frontend-desktop` 基础工程
2. 先完成 Desktop Shell 与 Window Manager，再迁移业务模块
3. 保持 V3 后端契约稳定，不把前端重构扩散成全栈重构
4. 灰度、降级、文档收口作为首批设计内容，不作为尾部补丁
